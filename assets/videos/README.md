# 视频文件说明

## 使用说明

请将测试视频文件放在此目录下，命名为 `sample.mp4`

## 视频要求

- 格式：MP4 (H.264 编码)
- 分辨率：建议 1280x720 或 1920x1080
- 时长：任意
- 大小：建议不超过 100MB

## 测试视频

如果没有测试视频，可以使用以下方式获取：

1. 使用自己的视频文件
2. 从网上下载免费测试视频
3. 使用浏览器录制屏幕

## 注意事项

- 确保视频文件路径正确：`assets/videos/sample.mp4`
- 如果视频无法播放，检查浏览器控制台错误信息
- 某些浏览器可能需要 HTTPS 才能播放视频
