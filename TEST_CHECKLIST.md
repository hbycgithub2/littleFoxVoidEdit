# 功能测试清单

## 🧪 测试步骤

### 1. 基础功能测试

#### 1.1 绘制热区
- [ ] 点击「圆形」按钮，在 Canvas 上拖拽绘制圆形
- [ ] 点击「矩形」按钮，在 Canvas 上拖拽绘制矩形
- [ ] 点击「椭圆」按钮，在 Canvas 上拖拽绘制椭圆
- [ ] 绘制时应该看到绿色预览线
- [ ] 释放鼠标后热区应该出现

#### 1.2 选择热区
- [ ] 点击热区，热区应该变成红色（选中状态）
- [ ] 点击空白区域，热区应该恢复绿色（取消选择）
- [ ] 按住 Ctrl 点击多个热区，多个热区同时变红
- [ ] 属性面板应该显示选中热区的信息

#### 1.3 拖拽移动
- [ ] 点击并拖拽热区，热区应该跟随鼠标移动
- [ ] 释放鼠标后，热区位置应该更新
- [ ] 拖拽时鼠标指针应该变成 pointer

#### 1.4 悬停效果
- [ ] 鼠标移到热区上，热区应该变成黄色
- [ ] 鼠标移开，热区应该恢复原色
- [ ] 鼠标指针应该变化

### 2. 编辑功能测试

#### 2.1 属性面板
- [ ] 选中热区后，属性面板应该显示
- [ ] 修改「单词」输入框，热区数据应该更新
- [ ] 修改「开始时间」，热区数据应该更新
- [ ] 修改「结束时间」，热区数据应该更新
- [ ] 修改「颜色」，热区颜色应该立即改变

#### 2.2 热区列表
- [ ] 创建热区后，列表应该显示新热区
- [ ] 点击列表项，对应热区应该被选中
- [ ] 删除热区后，列表应该更新

#### 2.3 删除功能
- [ ] 选中热区，点击「删除」按钮，热区应该消失
- [ ] 选中热区，按 Delete 键，热区应该消失
- [ ] 多选热区，删除应该删除所有选中的热区

### 3. 撤销/重做测试

#### 3.1 撤销功能
- [ ] 创建热区后，点击「撤销」按钮，热区应该消失
- [ ] 删除热区后，点击「撤销」按钮，热区应该恢复
- [ ] 移动热区后，点击「撤销」按钮，热区应该回到原位
- [ ] 按 Ctrl+Z，应该执行撤销

#### 3.2 重做功能
- [ ] 撤销后，点击「重做」按钮，操作应该恢复
- [ ] 按 Ctrl+Shift+Z，应该执行重做

### 4. 数据管理测试

#### 4.1 导出功能
- [ ] 创建几个热区
- [ ] 点击「导出」按钮
- [ ] 应该下载 hotspots.json 文件
- [ ] 打开 JSON 文件，检查数据格式是否正确

#### 4.2 导入功能
- [ ] 点击「导入」按钮
- [ ] 选择之前导出的 JSON 文件
- [ ] 热区应该正确加载
- [ ] 应该显示「导入成功」提示

### 5. 快捷键测试

- [ ] Ctrl+Z - 撤销
- [ ] Ctrl+Shift+Z - 重做
- [ ] Delete - 删除选中热区
- [ ] Esc - 取消绘制模式
- [ ] Ctrl+点击 - 多选

### 6. 视频功能测试（如果有视频文件）

- [ ] 视频应该正常播放
- [ ] Canvas 应该与视频对齐
- [ ] 点击「播放」按钮，视频应该播放
- [ ] 点击「暂停」按钮，视频应该暂停
- [ ] 热区应该根据视频时间显示/隐藏
